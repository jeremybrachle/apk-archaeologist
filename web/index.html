Real Reconstruction Errors (15)

These simulate what actually goes wrong when you decompile an APK and let AI fill the gaps
  • 3 legs — AI merged animation frames into simultaneous legs
  • 2 eyes instead of 1 — AI inferred bilateral symmetry from side-view code
  • Tail curves UP — sign flip in obfuscated Pair return value
  • Angular body — corner radius 3 vs 8, AI couldn't map which constant was which
  • Bug-eyed — eye radius 8 vs 6, constants swapped
  • Exaggerated squash/stretch — multiplier 0.35/0.30 vs 0.15/0.12, AI doubled it
  • Thin spindly legs — stroke 4px vs 6px
  • Eye position shifted — offset X/Y values swapped with other constants
  • Square clouds — lost borderRadius parameter in obfuscation
  • Dense oversized rocks — hash modulo 5 vs 7, base size 2 vs 1
  • 2× bird wing speed — scroll divisor 5 vs 10
  • Symmetric cactus arms — AI assumed mirror symmetry
  • Score 17% slower — timer 0.12s vs 0.1s, guessed from nearby floats
  • Birds spawn early — threshold 30 vs 50, wrong integer literal
  • Colors slightly off — hex strings partially corrupted in decompiled output
