Stage 1 → 2: What Extraction Does to the File Structure

The APK is a ZIP archive containing compiled Dalvik bytecode (DEX), compressed resources, and a binary manifest. JADX decompiles the DEX back to Java/Kotlin, and apktool decodes resources. ProGuard has already destroyed class names, method names, and inlined small helper classes.

Stage 2 → 3: What AI Inference Produces

The AI reads the decompiled output, maps obfuscated names back to likely originals, and generates reconstruction reports. It tries to rebuild stripped classes from the orphaned code fragments. Every guess is recorded with a confidence score.

Stage 3 → 4: From Inference to Runnable Unity Project

The AI inference output gets transformed into a Unity project structure. Android Canvas draw calls map to Unity sprites/meshes. Game logic maps to MonoBehaviours. The mutations from frog DNA carry over — this is the "new dinosaur" that walks and roars, but not quite like the original.

Stage 4 → 5: Unity Build to APK Patch

The Unity project compiles to a new APK, but instead of replacing the original, it generates a patch APK that runs alongside it. Think of it as a second-generation dinosaur — it inherits the original's mutations (from frog DNA) but each iteration can be refined, gradually evolving toward a stable, faithful reproduction.

Example 1: ParticleSystem — Completely Stripped

Original Kotlin
class ParticleSystem { data class Particle( var x: Float, var y: Float, var vx: Float, var vy: Float, var life: Float, val maxLife: Float, val radius: Float // 1.5f..3.5f ) fun spawnDust(x: Float, y: Float, n: Int) { repeat(n) { particles += Particle( x = x + random(-15f, 15f), y = y, vx = random(-30f, 30f), // horizontal spread vy = random(10f, 40f), // initial upward life = 0.4f, maxLife = 0.4f, radius = random(1.5f, 3.5f) ) } } fun update(dt: Float) { particles.forEach { it.vy += 600f * dt // gravity pulls DOWN it.x += it.vx * dt it.y += it.vy * dt it.life -= dt } } fun draw(canvas: Canvas) { particles.forEach { paint.alpha = (it.life/it.maxLife * 255).toInt() paint.color = 0xFF8B7355.toInt() // brown canvas.drawCircle(it.x, it.y, it.radius, paint) } } }
